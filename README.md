# 💰 ZigZag's Workshop
*"Built for those who value gear over safety."*

### Features
- **Workshop:** Public gadgets and mad engineering for everyone.
- **Black Market:** Restricted access. Requires an **Invitation** for The Black Market to enter and trade.
- **The Vault:** Personal inventory to store and view all your loot.
- **Economy:** Virtual currency balance management and top-ups via Stripe.
- **Lore-driven:** Dynamic UI themes based on your access level.
- **Custom UI:** A handcrafted design built from scratch to match the goblin atmosphere.

### Tech Stack
- **Frontend:** Vue 3 (Composition API), Quasar Framework, TypeScript.
- **State:** Pinia (Modular stores).
- **Backend/Auth:** Supabase (PostgreSQL, Edge Functions).
- **Payments:** Stripe for gold top-ups.
- **Animations:** GSAP for immersive UI.
- **Styling:** SCSS & Custom Themes.

### 📍 Location
Smuggler's Alley, Tinker Tangle.
